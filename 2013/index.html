About

在求學的過程中，可曾想過課堂所上的一切，究竟能發揮在哪？

延續 SITCON 學生計算機年會的理念，HackGen 駭客世代黑客松提供了絕佳的機會，讓身為學生的你我得以發揮自己所學，透過實際的程式撰寫，為彼此的校園生活點亮更多的驚嘆號！

透過 HackGen，我們希望能夠讓各方學生們都能體會黑客松的樂趣，並促進跨校、跨領域學生的認識以及交流；甚至藉由作品分享，讓每位參賽的學生，都能見識先前可能從未接觸的技術，學以致用、教學相長。

主題

本次活動的主題為「變法圖強」，取自清末維新運動中康有為提出的口號，希望能透過改善現有的制度，以健全社會的運作。在本活動中，我們希望讓學生著手改善自己在學習環境中缺少或是未臻完善的事物，並透過實作讓相關單位了解學生的訴求，以達到透過寫程式改造教育的目的。

作品類型

資訊類
統整校園相關資訊或規章，藉由程式處理、分析，降低學生存取的門檻。如：學校法規整理、獎學金資訊整理、活動資訊等。
平台類
提供學生分享學習、課程或是書籍等資訊，藉由建立平台使學生得以互相交流。如：二手書與文具交易平台、課業交流平台等。
系統類
設計校園環境中會使用到之資訊系統。如：選課模擬系統、課程即時回饋系統或是師生互動系統等。
增益類
以目前學校既有之資訊平台、系統加以改善，如美化界面、提昇使用經驗、資訊系統導覽等。
其他類
其他類任何能改善學生學習品質之議題。

競賽說明

  • 本活動每場限制最多五十人參與，報名後工作人員將於兩天之內以 E-Mail 的方式確認。
  • 本活動作品不限制任何平台及工具，但是作品之主題需要環繞學生議題為主（如：選課輔助、數位學習、考古題交流等），故參與本活動前需事先提供作品名稱及簡介給主辦單位審核。
  • 每隊禁止使用以完成之作品參加比賽，但可以事先準備所需之素材。
  • 本活動全程使用 github 系統繳交作品原始程式碼至指定的群組下，隊伍中的隊員需事先申請 github 帳號以方便作業。
  • 每隊需於最後上台展示成果，每隊展示 5 ~ 10 分鐘，並由評審提問。
  • 兩場次可重複報名，但不可使用相同題目。
